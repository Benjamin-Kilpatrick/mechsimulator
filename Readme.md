# MechSimulator
Mech simulator

## Setup
### Linux
Setup env
```bash
git clone https://github.com/Benjamin-Kilpatrick/mechsimulator.git
cd mechsimulator

python -m venv venv
source ./venv/bin/activate
pip install -r Requirements.txt
```
